# WorkOutApp 手動作業タスク

## 運用ルール

### 必須ルール（厳守）
1. **各Workには「必要なタイミング」を記載（progressのどのPhase開始前に必要か）**
2. **タスク完了時は該当タスクに [完了: YYYY-MM-DD] を追記**
3. **Workの全タスク完了後、必ず該当Workをarchive.txtに移動する**
   - 移動を忘れた場合、次のWorkまたは関連Phaseに進むことは禁止
   - 移動前に全タスクが [完了] になっていることを確認
4. **問題発生時は必ず problem.txt に記録**

### archive移動手順
1. Work内の全タスクに [完了: YYYY-MM-DD] が付いていることを確認
2. 該当Workのセクション全体をコピー
3. archive.txt の「完了済み Work」セクションに貼り付け
4. manual-work.txt から該当Workを削除
5. archive.txt に完了日を記録

---

## 作業順序ガイド

推奨される作業順序（progressのPhaseと並行して進める）:

```
Manual Work 1 → Manual Work 2 → Phase 1〜3
                                    ↓
                               Phase 4〜8
                                    ↓
               Manual Work 3 → Phase 9（課金）
               Manual Work 4 ↗
                                    ↓
                               Phase 10（通知）
                                    ↓
                               Phase 11
                                    ↓
                               Phase 12（テスト）
                                    ↓
               Manual Work 5 → Phase 13（ビルド準備）
                                    ↓
               Manual Work 6 → Phase 14（公開）
               Manual Work 7 ↗
```

## Work 3: APNs 証明書設定
必要なタイミング: **Phase 10 開始前**

### 3.1 APNs 認証キー作成
- [ ] Apple Developer Portal（https://developer.apple.com/）にアクセス
- [ ] Certificates, Identifiers & Profiles → Keys
- [ ] 新規キー作成
  - [ ] Key Name: WorkOutApp APNs Key
  - [ ] Apple Push Notifications service (APNs) にチェック
- [ ] キーファイル（.p8）ダウンロード
- [ ] Key ID 記録
- [ ] Team ID 記録

### 3.2 Firebase への APNs 設定
- [ ] Firebase Console → プロジェクト設定 → Cloud Messaging
- [ ] APNs認証キーをアップロード
  - [ ] .p8 ファイルアップロード
  - [ ] Key ID 入力
  - [ ] Team ID 入力

### 3.3 App ID 設定
- [ ] Apple Developer Portal → Identifiers
- [ ] App ID 作成または編集
  - [ ] Bundle ID: com.yourcompany.workoutapp
  - [ ] Push Notifications を有効化

## Work 4: In-App Purchase 商品登録
必要なタイミング: **Phase 9 開始前**

### 4.1 App Store Connect アプリ作成
- [ ] App Store Connect（https://appstoreconnect.apple.com/）にアクセス
- [ ] 新規アプリ作成
  - [ ] プラットフォーム: iOS
  - [ ] アプリ名: WorkOutApp（仮）
  - [ ] プライマリ言語: 日本語
  - [ ] Bundle ID: 選択
  - [ ] SKU: workoutapp001（任意）

### 4.2 App 内課金商品登録
- [ ] App Store Connect → アプリ → App 内課金
- [ ] 月額サブスクリプション作成
  - [ ] 参照名: 月額プレミアム
  - [ ] 製品ID: com.workoutapp.subscription.monthly
  - [ ] サブスクリプショングループ作成
  - [ ] 価格: ¥500
  - [ ] ローカライズ情報入力
- [ ] 年額サブスクリプション作成
  - [ ] 参照名: 年額プレミアム
  - [ ] 製品ID: com.workoutapp.subscription.yearly
  - [ ] 同じサブスクリプショングループに追加
  - [ ] 価格: ¥5,000
  - [ ] ローカライズ情報入力

### 4.3 Shared Secret 取得
- [ ] App Store Connect → アプリ → App 内課金 → App 共有シークレット
- [ ] 共有シークレット生成
- [ ] シークレットを Firebase Functions 環境変数に設定

### 4.4 Sandbox テスター作成
- [ ] App Store Connect → ユーザとアクセス → Sandbox → テスター
- [ ] テスター追加（テスト用メールアドレス）

# WorkOutApp 手動作業タスク

## 運用ルール

### 必須ルール（厳守）
1. **各Workには「必要なタイミング」を記載（progressのどのPhase開始前に必要か）**
2. **タスク完了時は該当タスクに [完了: YYYY-MM-DD] を追記**
3. **Workの全タスク完了後、必ず該当Workをarchive.txtに移動する**
   - 移動を忘れた場合、次のWorkまたは関連Phaseに進むことは禁止
   - 移動前に全タスクが [完了] になっていることを確認
4. **問題発生時は必ず problem.txt に記録**

### archive移動手順
1. Work内の全タスクに [完了: YYYY-MM-DD] が付いていることを確認
2. 該当Workのセクション全体をコピー
3. archive.txt の「完了済み Work」セクションに貼り付け
4. manual-work.txt から該当Workを削除
5. archive.txt に完了日を記録

---

## 作業順序ガイド

推奨される作業順序（progressのPhaseと並行して進める）:

```
Manual Work 1 → Manual Work 2 → Phase 1〜3
                                    ↓
                               Phase 4〜8
                                    ↓
               Manual Work 3 → Phase 9（課金）
               Manual Work 4 ↗
                                    ↓
                               Phase 10（通知）
                                    ↓
                               Phase 11
                                    ↓
                               Phase 12（テスト）
                                    ↓
               Manual Work 5 → Phase 13（ビルド準備）
                                    ↓
               Manual Work 6 → Phase 14（公開）
               Manual Work 7 ↗
```

## Work 7: App Store 審査提出
必要なタイミング: **Phase 14 と並行して実施**

- [ ] サポートURL
- [ ] プライバシーポリシーURL

- [ ] App Store Connect → アプリ → App Store → ローカライズ追加
- [ ] English (U.S.) を追加
- [ ] スクリーンショット準備・アップロード（英語テキスト用）

  - [ ] 5.5インチ
